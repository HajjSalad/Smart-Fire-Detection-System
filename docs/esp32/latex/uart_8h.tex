Initialize UART0 with default settings. 

Must be called before performing any UART communication.

Initialize UART0 with default settings.

This function configures UART0 with 8 data bits, 1 stop bit, no parity, no flow control, and a baud rate of 115200. It also sets the TX and RX pins, installs the UART driver, and allocates RX and TX buffers.
