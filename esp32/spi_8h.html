Detailed Description

Interface for SPI Master initialization on ESP32 using ESP-IDF.

This header file declares the SPI initialization function and external handles required to manage SPI communication.

void spi_init ( )

Initialize the SPI bus and attach the SPI device.

This function must be called before any SPI transactions are performed. It sets up the SPI bus and adds the SPI device to the bus with the specified configuration.

Initialize the SPI bus and attach the SPI device.

This function sets up thew SPI bus with specified parameters and add the SPI device to the bus using the ESP-IDF SPI Master driver.

  • Clock speed: 100 kHz
  • Mode: SPI Mode 0 (CPOL = 0, CPHA = 0)
  • SS pin: GPIO 5
  • Max transfer size: 64 bytes
